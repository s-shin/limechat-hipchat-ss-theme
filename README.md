
概要
----

HipChatライクなLimeChatテーマ。

日本語をクエリに含むエンコード済みのURLをデコードして見やすくするスクリプトを同伴。

インストール
------------

```
cd "~/Library/Application Support/net.limechat.LimeChat-AppStore/Themes/"
git clone https://github.com/s-shin/limechat-hipchat-ss-theme.git
rake install
```
